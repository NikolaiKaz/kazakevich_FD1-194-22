Безопасная палитра

Безопасная палитра цветов – это общепринятый стандарт для web-дизайнеров. Безопасная цветовая палитра - это набор из 216 цветов, общих для всех инструментальных платформ. Каждый из них может быть задан однобайтовым кодом.

Для чего используется безопасная палитра:

  • цвета палитры отображаются одинаково на всех платформах и браузерах.
  • цвета палитры корректно отображаются на старых компьютерах.
  • цвета палитры можно использовать для сосздания изображения формата GIF, так так для этого досточно 216 цветов.
  • цветовая палитра является приоритетной при выборе цвета для фона или цвета шрифта
  • цвета палитры необходимо использовать для создания изображений.
